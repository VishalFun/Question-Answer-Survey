Question Answer Survey App

This app saves user response according to asked question. And the response is send to admin through mail in pdf.

Requirement:
Python > 3.5

(install this package using pip)

django 		
crispy_forms
xhtml2pdf

how to use:
1. Open cmd in current folder.
2. Run cmd python manage.py makemigrations
3. Run cmd python manage.py migrate
4. Run cmd python manage.py runserver. Paste http://localhost:8000/ on browser.

Enjoy!!!!!!
